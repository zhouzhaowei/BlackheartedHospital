# BlackheartedHospital

网传附莆田系医院名单，欢迎更新,维护这份名单,本人不会有任何商业目的,过去不会有，将来也不会有，如果出现不当的情况，请大家及时指出，随时监督。但是本人不会保证别人有商业目地或其它动机,在维护的过程中会尽最大努力避免。

# 版本1.2beta

- 1.紧急更新来自凤凰网的数据 <http://news.ifeng.com/mainland/special/ptxyy/>
- 2.大约500家，和前面有重复的数据
- 3.这份数据来自网络，并不完全准确，如果有写错了的,请及时提出来或提交更新(请提交到 README.md 文件)
- 4.@wandergis 已经根据凤凰网的数据做了一个地图出来 https://github.com/wandergis/putian-hospitals
- 5.为什么不推荐做原生的APP，比如说Android的APK包，因为要验证一个APK包是否很麻烦，现在的功能只是一个简单的查询功能，网页查询很轻松的满足了，验证一个APK包，权限是不是有问题，比如说要通讯录的权限等等的，还要抓包检查是否有收集用户信息等（比如说原生的APK做一个功能，用户第三次打开的时候，才去收集用户的手机信息，这个就很难发现），所以这里不推荐做原生的APP
- 6.以上的数据来自网络，现在并不只包括莆田系的医院，我希望这是一个中性的名单，项目里面不做评价，这里只是大家的一个收集，并不代表真的有问题，用户看到之后自己评价，至于用户怎么理解，并不干涉，我也不相信所以的莆田系医院都有问题（莆田系市场份额具说至80%），还有很多医院只是外包出去，外包出去并不一定就有问题，这样的逻辑大家都应该明白。就算是有问题的，能够及时发现并改正，这也算是一件美好的事情。
- 7.trending里面排名第一，我开始还是很高兴的，后来很紧张，并不是我的本意，只是star的太多了，所以就排上去了，我建这个工程的目的不是为了trending排到前面去。是希望这个世界更加美好一点。
- 8.那些说github要被墙的人，希望你能对自己的祖国有点信心好不好。不要这么内心黑暗，现在媒体都在报道，说明这个事情大家都在重视起来，将来医疗环境肯定比现在好。
